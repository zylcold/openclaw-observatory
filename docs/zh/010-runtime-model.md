# 运行时模型

```text
宿主机
└── OpenClaw 实例
    ├── Gateway
    ├── 工作区
    ├── 会话
    │   └── Agent 运行
    │       ├── LLM 调用
    │       ├── 工具调用
    │       ├── MCP 调用
    │       └── 子 Agent 运行
    └── 资源采样
```

| 对象 | 定义和生命周期 | 标识和父级 | 主要属性 | Prometheus 标签？ | 持久化？ |
| --- | --- | --- | --- | --- | --- |
| 宿主机 | 运行守护进程的机器；跨 Gateway 重启存在 | 本地派生的匿名宿主 ID；根节点 | OS、架构 | 无原始宿主 ID | 是，通过实例元数据 |
| 实例 | 一个 OpenClaw 状态/配置安装 | 稳定的哈希实例 ID；父级为宿主机 | 适配器版本、最后可见时间、当前 PID | `instance`，仅有界时 | 是 |
| Gateway | 一次 OS 进程生命周期 | `(instanceId, processId, processStart)` | 启动/停止/崩溃、版本、运行时间 | 仅 instance | 是，通过生命周期事件表示 |
| 工作区 | Agent 工作上下文 | 哈希/规范化的工作区标识；父级为实例 | 仅安全名称/哈希 | 否 | 后续 Schema 中可选 |
| 会话 | OpenClaw 对话/运行时会话 | OpenClaw `sessionId`；父级为实例 | 状态、开始/结束/原因、消息数 | 否 | 是 |
| Agent 运行 | 一次接受的 Agent 轮次 | OpenClaw `runId`；父级为会话 | 提供商、模型、状态、持续时间、触发器 | 否 | 是 |
| LLM 调用 | 运行内的一次提供商尝试 | OpenClaw `callId`；父级为 Agent 运行 | 提供商、模型、计时、结果、Token/成本核算 | 仅 provider/model/status | 是 |
| 工具调用 | 一次工具执行 | `toolCallId`，或确定性事件派生回退；父级为 Agent 运行 | 规范化的工具、来源、持续时间、结果 | 有界 tool/status | 是 |
| MCP 调用 | 来源报告为 `mcp` 的工具调用 | 相同标识规则；父级为 Agent 运行 | 安全可用的 server/owner、持续时间、结果 | 有界 tool/status | 是 |
| 子 Agent 运行 | Agent 启动的子会话/运行 | 子会话哈希 + 运行 ID；父级为 Agent 运行/会话 | 模式、子 Agent、模型、结果 | 否 | 是 |
| 资源采样 | 即时进程测量 | 生成的事件 ID；父级为 Gateway/实例 | CPU 时间、RSS、VM、线程、FD、IO | 仅 instance | 是，降采样后 |

## 必要区分

- **实例**跨进程重启存活；**Gateway** 是一次进程生命周期。
- **会话**是长期的 OpenClaw 状态边界；**Agent 运行**是一次轮次。
- **LLM 调用**是一次提供商尝试。重试/故障转移可在一次运行内产生多次调用。
- **工具调用**是语义执行；**MCP 调用**是其中来源被权威报告为 MCP 的子集。
- **资源采样**是守护进程观察的宿主数据，可在没有活跃会话时独立存在。

## 标识符处理

原始会话密钥可能嵌入路由身份，在插件中进行哈希处理。OpenClaw 提供的会话/运行/调用 ID 用于本地关联但从不作为指标标签。PID 本身不足以作为标识，因为操作系统会复用 PID；归约器将其与实例和进程启动证据配对使用。
